Chapter 2

What are we learning?

So yes, we’re learning to program, but what’s that really mean? What’s in store for our future?

All kinds of programming shares a little bit of the same tools. This includes buzzwords like variables, loops, math, libraries, blah blah blah. Whether you’re doing an analysis of ten million court records or building a street racing game, you’re going to come up against these concepts again and again. So we’ll start with those. They’re boring, but they’re kind of required for anything fun.

We’ll be learning this general programming stuff using the Python programming language. Many programming languages are especially good at one thing - microchips! gaming! mobile apps! statistics! - but Python is pretty well-rounded and can more or less do everything. Whatever you want to do, Python can prrrrobably help you out, so it’s a great one to learn.

Python is also super popular, and getting more popular all the time, which means you’ll find a lot of people online who can help you out when you run into problems. Weirdly, it was invented-ish way back in 19891, and I don’t know why it took so long to become the hot thing.

So, what’s next after you learn these basics? Well, the answer is whatever you want.

Programming is an almost-infinite amount of different stuff. There are entire realms of programming I’ve never even looked at! That’s no big deal, though, because I know that when I need those things (if I ever do), I can simply go learn them. After you know the basic concepts of programming, building what you want to build just becomes a game of figuring out what you need to know, and figuring out how to learn it.

But how do you learn this mysterious new programming stuff? You’ll spend time searching online, reading other peoples’ questions and answers. You’ll adapt code you find, leaning on people you know - and people you don’t know! - for answers and tips. When you start programming it isn’t something you go through alone; the internet and everyone on it becomes your new best friends.

Here is a promise, 100% guaranteed: whatever problem or question you might have, someone else has had that exact same question in the past week, or the past month, or the past year - and they’ve asked for a solution online, and then someone’s helped them. You just need to track that answer down!

Since that part is so important, we’ll definitely be spending time on learning how to search for answers and understand them. But I mean, you know how the internet can be2, so maybe we’ll save that until we have the basics down.

Anyway, I just hope you’ll work your way through this without feeling discouraged or too bored, because that’s all I see when I read through other coding tutorials. I try to take things nice and slow because I believe a tiny baby kitten could learn to program if they’re taught correctly!

If something isn’t clicking in your brain, I promise that it isn’t you, it’s me! If you find anything unclear or you decide to give up, please let me know so I can try to improve things.
