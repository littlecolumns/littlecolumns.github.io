Logistic Regression

Logistic regression is a technique to figure out how the relationship between two or more things. For example, how do SAT scores affect if you're accepted into college? You use logistic regression when the thing being predicted is a category, like accepted vs. not accepted. If you'd like to predict a number instead, try linear regression. Logistic regression it also often used to make comparisons - for example, if our SAT scores were the same, is it better to play a musical instrument or be on a sports team?
