Topics in DS4J

Topics for study

If you'd like to go all in on a "complete" data science curriculum, you might be interested in the Curriculum page, where I walk you through one possible option.

Natural Language Processing

General text analysis.

  1. Start by learning to count words simply or a more complex way.
  2. Skip over overly simplistic airbag analysis and go right to either sentiment analysis or app reviews. Stop by the crime classifier if you have time or a particular interest in classification.
  3. Shift to detecting special interest legislation, which will teach you all about n-grams and converting PDFs and Word docs to text
  4. Relax with a little more sentiment analysis, but a slightly more complex (and troubled?) version

How X affects Y

Featuring both linear and logistic regression.

  1. Learn the basics of linear regression, then apply the "change in X, change in Y" concept with the Associated Press or the Milwaukee Journal-Sentinel.
  2. Use the regression predictions for school-based investigations alongside the Dallas Morning News or The Tampa Bay Times.
  3. Pivot from linear regression (numeric output) to logistic regression (yes/no or categorical output) with ticketing, mortgages, or jury selection.
  4. Your capstone is ProPublica's COMPAS analysis.

I don't want to read a thousand documents

Very similar to natural language processing, but a bit more focused on classification.

  1. Start by learning to count words.
  2. Learn from others' mistakes with the NYT's airbag analysis, then level up to app reviews.
  3. Finally, the crime classifier will cover all the extra bases you might need.

Hi, welcome to Data Science for Journalism!

There's been a lot of buzz about machine learning and "artificial intelligence" being used in stories over the past few years. It's mostly not that complicated - a little stats, a classifier here or there - but it's hard to know where to start without a little help.

Hopefully this site can be that help! Learn more about this project here.
